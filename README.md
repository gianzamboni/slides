# Welcome to [Slidev](https://github.com/slidevjs/slidev)!

To start the slide show enter to the folder and run:

- `pnpm install`
- `pnpm dev`
- visit <http://localhost:3030>

Edit the [slides.md](./slides.md) to see the changes. If you use VSCode, you can use the extension [Slidev](https://open-vsx.org/extension/antfu/slidev) to have full integration with the editor.

Learn more about Slidev at the [documentation](https://sli.dev/).
